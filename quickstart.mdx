This guide gets you to a running simulation as fast as possible. You will install JSBSim, load a bundled aircraft script, execute the simulation loop, and read state back from the running simulation.

## Python quickstart

<Steps>
  <Step title="Install JSBSim">
    Install the Python package from PyPI. The wheel includes aircraft data and example scripts.

    ```bash
    pip install jsbsim
    ```
  </Step>

  <Step title="Verify the install">
    Confirm the package is installed and locate the bundled aircraft data:

    ```python
    import jsbsim

    print(jsbsim.__version__)
    print(jsbsim.get_default_root_dir())
    ```

    `get_default_root_dir()` returns the path to the directory that contains the `aircraft/`, `engines/`, and `scripts/` folders installed alongside the wheel.
  </Step>

  <Step title="Run a simulation">
    Create an `FGFDMExec` instance, load one of the bundled scripts, and step through the simulation:

    ```python
    import jsbsim

    fdm = jsbsim.FGFDMExec(None)  # None uses the default aircraft data directory
    fdm.load_script('scripts/c1723.xml')
    fdm.run_ic()

    while fdm.run():
        pass
    ```

    Passing `None` to `FGFDMExec` tells JSBSim to use the data installed with the wheel. The script `scripts/c1723.xml` is one of the many example scripts included by default.
  </Step>

  <Step title="Read simulation state">
    Use `get_property_value()` to read any property from the simulation tree during or after the loop:

    ```python
    import jsbsim

    fdm = jsbsim.FGFDMExec(None)
    fdm.load_script('scripts/c1723.xml')
    fdm.run_ic()

    while fdm.run():
        altitude = fdm.get_property_value('position/h-sl-ft')
        airspeed = fdm.get_property_value('velocities/vt-fps')
        print(f"Alt: {altitude:.0f} ft  Airspeed: {airspeed:.1f} fps")
    ```

    Properties follow the path convention `subsystem/property-name`. See the [property system reference](/concepts/property-system) for a full listing.
  </Step>
</Steps>

## Command-line quickstart

If you installed JSBSim through the Windows installer or Ubuntu packages, you can run a simulation directly from the shell.

<Steps>
  <Step title="Navigate to the data directory">
    Change to the root of the aircraft data package (the directory that contains `aircraft/`, `engines/`, and `scripts/`).
  </Step>

  <Step title="Run a script">
    Pass a script path to `JSBSim.exe` (Windows) or `JSBSim` (Linux):

    ```bash
    JSBSim.exe --script=scripts/c1721.xml
    ```

    JSBSim will execute the script in batch mode and print output to the console.
  </Step>

  <Step title="See all options">
    ```bash
    JSBSim.exe --help
    ```
  </Step>
</Steps>

## Complete runnable example

The following self-contained script runs a simulation and prints altitude and airspeed at each time step:

```python
import jsbsim

# Create the FDM executive using the default (wheel-bundled) aircraft data.
fdm = jsbsim.FGFDMExec(None)

# Load an example script. c1723.xml flies the c172 aircraft.
fdm.load_script('scripts/c1723.xml')

# Apply initial conditions from the script.
fdm.run_ic()

# Step the simulation until the script signals completion.
while fdm.run():
    t   = fdm.get_property_value('simulation/sim-time-sec')
    alt = fdm.get_property_value('position/h-sl-ft')
    vt  = fdm.get_property_value('velocities/vt-fps')
    print(f"t={t:6.1f}s  alt={alt:8.0f}ft  vt={vt:6.1f}fps")
```

## Available example scripts

The wheel includes several ready-to-run scripts. Commonly used ones include:

| Script | Description |
|---|---|
| `scripts/c1721.xml` | Cessna 172 takeoff and flight |
| `scripts/c1723.xml` | Cessna 172 cruise |

To list all installed scripts, inspect the `scripts/` subdirectory of the default root directory:

```python
import os
import jsbsim

scripts_dir = os.path.join(jsbsim.get_default_root_dir(), 'scripts')
for name in sorted(os.listdir(scripts_dir)):
    print(name)
```

## C++ quickstart

If you are embedding JSBSim in a C++ application, the minimal entry point is:

```cpp
#include <FGFDMExec.h>

int main(int argc, char **argv)
{
    JSBSim::FGFDMExec FDMExec;
    FDMExec.LoadScript(SGPath(argv[1]));
    FDMExec.RunIC();
    bool result = true;
    while (result) result = FDMExec.Run();
}
```

See the [C++ API guide](/integrations/cpp-api) for the full API surface.
