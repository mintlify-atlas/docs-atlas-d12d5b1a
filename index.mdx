JSBSim is a nonlinear 6 Degree-of-Freedom (6-DoF) Flight Dynamics Model (FDM) written in C++. It models the forces and moments acting on an aircraft — aerodynamics, propulsion, landing gear, buoyancy — and integrates the equations of motion to produce a complete flight simulation. JSBSim ships with Python, MATLAB/Simulink, and Unreal Engine bindings, and is used in everything from academic research to DARPA's AlphaDogfight Trials.

## Get started in three steps

<Steps>
  <Step title="Install JSBSim">
    Install the Python package from PyPI — aircraft data and example scripts are included:

    ```bash
    pip install jsbsim
    ```
  </Step>
  <Step title="Run a simulation">
    Load one of the bundled aircraft and execute the simulation loop:

    ```python
    import jsbsim

    fdm = jsbsim.FGFDMExec(None)
    fdm.load_script('scripts/c1723.xml')
    fdm.run_ic()

    while fdm.run():
        pass
    ```
  </Step>
  <Step title="Inspect simulation state">
    Read any property from the running simulation tree:

    ```python
    altitude = fdm.get_property_value('position/h-sl-ft')
    airspeed = fdm.get_property_value('velocities/vt-fps')
    print(f"Altitude: {altitude:.0f} ft, Airspeed: {airspeed:.1f} fps")
    ```
  </Step>
</Steps>

## Key capabilities

- **Nonlinear 6-DoF dynamics** — Full equations of motion with Coriolis and centrifugal effects
- **XML-driven configuration** — Aircraft, engines, flight control systems, and scripts are fully defined in XML
- **Accurate Earth model** — WGS84 geodetic coordinates, oblate spheroid shape, ISA 1976 atmosphere
- **Rich propulsion models** — Piston, turbine, turboprop, rocket, electric motors, and rotors
- **Configurable FCS** — PID controllers, actuators, filters, sensors, switches, and more
- **Multiple output formats** — CSV, socket, FlightGear native protocol, and custom formats
- **Battle-tested** — Verified by NASA; used in ArduPilot, PX4 SITL, and DARPA research
