---
title: "Propulsion engines"
description: "Reference for all engine, thruster, and tank types supported in JSBSim aircraft XML files, including configuration parameters and exposed state properties."
---

JSBSim's propulsion system is declared in the `<propulsion>` section of an aircraft XML file. Each `<engine>` element references a separate engine definition file and a `<thruster>` element references the matching thruster file (propeller, nozzle, or rotor).

```xml
<propulsion>

  <engine file="a_piston_engine">
    <location unit="IN">
      <x>  28.0 </x>
      <y>   0.0 </y>
      <z> -12.4 </z>
    </location>
    <orient unit="DEG">
      <roll>  0 </roll>
      <pitch> 0 </pitch>
      <yaw>   0 </yaw>
    </orient>
    <feed> 0 </feed>   <!-- tank index to draw fuel from -->
    <thruster file="a_propeller">
      <sense> 1 </sense>       <!-- 1=CW, -1=CCW as seen from cockpit -->
      <p_factor> 6.0 </p_factor>
      <location unit="IN">
        <x> 28.0 </x>
        <y>  0.0 </y>
        <z>-12.4 </z>
      </location>
    </thruster>
  </engine>

  <tank type="FUEL">
    <location unit="IN">
      <x> 0 </x><y>  18 </y><z> -8 </z>
    </location>
    <capacity unit="LBS"> 114 </capacity>
    <contents unit="LBS">  57 </contents>
    <type> 1 </type>   <!-- 1=mogas/avgas, 2=Jet-A -->
  </tank>

</propulsion>
```

## Engine types

### FGPiston — reciprocating piston engine

**XML element (engine definition file root):** `<piston_engine>`

Models a supercharged piston engine based on Dave Luff's thermodynamic equations. Throttle position controls manifold pressure; power follows a volumetric-efficiency curve that degrades with altitude.

<AccordionGroup>
  <Accordion title="Configuration parameters">

```xml
<piston_engine name="IO-360-C1F6">
  <minmp unit="INHG"> 6.5  </minmp>
  <maxmp unit="INHG"> 28.5 </maxmp>
  <idlerpm>  700  </idlerpm>
  <maxrpm>  2700  </maxrpm>
  <maxhp unit="HP"> 180 </maxhp>
  <displacement unit="IN3"> 360 </displacement>
  <cycles>  4 </cycles>
  <bore   unit="IN"> 5.125 </bore>
  <stroke unit="IN"> 4.375 </stroke>
  <cylinders>  4 </cylinders>
  <compression-ratio> 8.5 </compression-ratio>
  <sparkfaildrop>   0.1  </sparkfaildrop>
  <static-friction unit="HP"> 1.5 </static-friction>
  <bsfc unit="LBS/HP*HR"> 0.45 </bsfc>
  <volumetric-efficiency> 0.85 </volumetric-efficiency>
  <!-- supercharger boost speeds (up to 3) -->
  <numboostspeeds>  1 </numboostspeeds>
  <ratedboost1 unit="INHG"> 29.5 </ratedboost1>
  <ratedpower1 unit="HP">  180  </ratedpower1>
  <ratedrpm1>   2700  </ratedrpm1>
  <ratedaltitude1 unit="FT"> 0 </ratedaltitude1>
</piston_engine>
```

| Parameter | Unit options | Description |
|---|---|---|
| `minmp` | INHG, PA, ATM | Idle manifold pressure at sea level |
| `maxmp` | INHG, PA, ATM | Maximum manifold pressure at sea level |
| `idlerpm` | — | Idle RPM; engine stops below 80% of this value |
| `maxrpm` | — | Maximum rated RPM |
| `maxhp` | HP, WATTS | Rated power at `maxrpm` |
| `displacement` | IN3, LTR, CC | Engine displacement |
| `bsfc` | LBS/HP*HR, KG/KW*HR | Brake-specific fuel consumption |
| `volumetric-efficiency` | — | Volumetric efficiency (0–1) |
| `compression-ratio` | — | Affects altitude efficiency degradation |
| `numboostspeeds` | — | Number of supercharger speeds (max 3) |

  </Accordion>
  <Accordion title="State properties">

Properties are at `propulsion/engine[n]/`:

| Property | Description |
|---|---|
| `power-hp` | Current brake horsepower |
| `engine-rpm` | Engine (crankshaft) RPM |
| `fuel-flow-rate-gph` | Fuel flow (gallons per hour) |
| `mp-osi` | Manifold pressure (inHg) |
| `egt-degF` | Exhaust gas temperature |
| `cht-degF` | Cylinder head temperature |
| `running` | 1 if engine is running |
| `cranking` | 1 if starter is engaged |

  </Accordion>
</AccordionGroup>

### FGTurbine — jet / turbofan engine

**XML element:** `<turbine_engine>`

Table-driven turbine model. Thrust is computed from lookup tables (`IdleThrustLookup`, `MilThrustLookup`, `MaxThrustLookup`) indexed by Mach and altitude. Supports afterburner and water injection.

<AccordionGroup>
  <Accordion title="Configuration parameters">

```xml
<turbine_engine name="GE-CF6-80C2">
  <milthrust unit="LBS"> 56750 </milthrust>
  <maxthrust unit="LBS"> 61960 </maxthrust>   <!-- with afterburner -->
  <bypassratio>   5.05 </bypassratio>
  <bleed>         0.04 </bleed>
  <tsfc>          0.62 </tsfc>                <!-- lbm/hr/lbf at cruise -->
  <atsfc>         1.80 </atsfc>               <!-- afterburner TSFC -->
  <idlen1>       30.0  </idlen1>
  <idlen2>       60.0  </idlen2>
  <maxn1>       100.0  </maxn1>
  <maxn2>       100.0  </maxn2>
  <n1spinup>      1.0  </n1spinup>
  <n2spinup>      3.0  </n2spinup>
  <n1startrate>   1.4  </n1startrate>
  <n2startrate>   2.0  </n2startrate>
  <n1spindown>    2.0  </n1spindown>
  <n2spindown>    2.0  </n2spindown>
  <augmented>     1    </augmented>
  <augmethod>     1    </augmethod>
  <injected>      0    </injected>
  <disable-windmill> 0 </disable-windmill>
</turbine_engine>
```

| Parameter | Description |
|---|---|
| `milthrust` | Static thrust at sea level, military (unaugmented) power |
| `maxthrust` | Static thrust with afterburner engaged |
| `bypassratio` | Bypass ratio (used for spool-up time estimation) |
| `bleed` | Fractional thrust reduction due to bleed losses |
| `tsfc` | Thrust-specific fuel consumption (lbm/hr/lbf) |
| `atsfc` | Afterburner TSFC |
| `idlen1` / `idlen2` | Fan / core rotor % at idle |
| `maxn1` / `maxn2` | Fan / core rotor % at full throttle |
| `augmented` | `1` if afterburner installed |
| `augmethod` | `0` = property, `1` = last 1% of throttle, `2` = FCS-defined |
| `injected` | `1` if water injection installed |
| `disable-windmill` | Prevent windmilling when engine is off |

  </Accordion>
  <Accordion title="Operating phases">

| Phase | Description |
|---|---|
| `tpOff` | Engine is not running |
| `tpRun` | Normal operation |
| `tpSpinUp` | Starter spinning engine |
| `tpStart` | Fuel lit, accelerating to idle |
| `tpStall` | Compressor stall |
| `tpSeize` | Inner spool seized |
| `tpTrim` | Simplified thrust for trim |

  </Accordion>
  <Accordion title="State properties">

Properties at `propulsion/engine[n]/`:

| Property | Description |
|---|---|
| `n1` | Fan rotor speed (%) |
| `n2` | Core rotor speed (%) |
| `thrust-lbs` | Net thrust (lbf) |
| `fuel-flow-pph` | Fuel flow (lb/hr) |
| `epr` | Engine pressure ratio |
| `egt-degc` | Exhaust gas temperature (°C) |
| `oil-pressure-psi` | Oil system pressure |
| `augmentation` | Afterburner engaged flag |
| `reverser` | Thrust reverser engaged flag |

  </Accordion>
</AccordionGroup>

### FGTurboProp — turboprop engine

**XML element:** `<turboprop_engine>`

Turboprop model that drives a propeller through a gearbox. Power is available to the propeller shaft as a function of N1 and atmospheric conditions.

<AccordionGroup>
  <Accordion title="Key configuration parameters">

```xml
<turboprop_engine name="Walter M601">
  <milthrust unit="LBS"> 0     </milthrust>   <!-- shaft power, not jet thrust -->
  <idlen1>    52.0 </idlen1>
  <maxn1>    100.0 </maxn1>
  <maxpower unit="HP"> 750 </maxpower>
  <psfc>     0.6   </psfc>   <!-- pph/HP at 100% N1 -->
  <n1idle_max_delay> 5 </n1idle_max_delay>   <!-- N1 time constant -->
  <betarangeend>    12 </betarangeend>        <!-- % throttle for beta range -->
  <reversemaxpower>  0 </reversemaxpower>     <!-- % max power in reverse -->
  <maxstartenginetime> 20 </maxstartenginetime>
  <startern1>         25 </startern1>
  <ielumaxtorque>   1000 </ielumaxtorque>
  <itt_delay>          3 </itt_delay>
</turboprop_engine>
```

| Parameter | Description |
|---|---|
| `maxpower` | Maximum shaft power |
| `psfc` | Power-specific fuel consumption (pph/HP) |
| `betarangeend` | Throttle % below which engine is at idle and propeller pitch is beta-range controlled |
| `ielumaxtorque` | Torque limit for the integrated electronic limiter unit |
| `n1idle_max_delay` | Time constant for N1 changes (seconds) |

  </Accordion>
  <Accordion title="State properties">

| Property | Description |
|---|---|
| `n1` | Compressor speed (%) |
| `itt-degc` | Inter-turbine temperature (°C) |
| `power-hp` | Shaft power (HP) |
| `fuel-flow-pph` | Fuel flow (lb/hr) |

  </Accordion>
</AccordionGroup>

### FGRocket — rocket engine

**XML element:** `<rocket_engine>`

Liquid or solid rocket. For liquid rockets, thrust is computed from specific impulse and mass flow. For solid rockets, a normalized thrust-vs-time table is provided.

<AccordionGroup>
  <Accordion title="Configuration and parameters">

```xml
<rocket_engine name="SRB">
  <isp>  280 </isp>              <!-- vacuum specific impulse, seconds -->
  <builduptime> 0.5 </builduptime>
  <maxthrottle> 1.0 </maxthrottle>
  <minthrottle> 0.0 </minthrottle>
  <slfuelflowmax unit="LBS/SEC"> 850 </slfuelflowmax>
  <sloxiflowmax  unit="LBS/SEC">   0 </sloxiflowmax>
  <PropellantType> SOLID </PropellantType>

  <!-- Normalized thrust profile for solid rocket: -->
  <thrust_table name="propulsion/thrust_time" type="internal">
    <tableData>
       0.0   0.00
       0.2   0.91
      16.0   0.99
     120.0   0.09
     132.0   0.00
    </tableData>
  </thrust_table>
</rocket_engine>
```

| Parameter | Description |
|---|---|
| `isp` | Vacuum specific impulse (seconds) |
| `slfuelflowmax` | Sea-level propellant mass flow at full throttle |
| `sloxiflowmax` | Sea-level oxidizer flow (liquid bi-prop only) |
| `builduptime` | Time for thrust to ramp from 0 to full (seconds) |
| `thrust_table` | Normalized thrust as a function of burn time (solid motors) |

  </Accordion>
  <Accordion title="State properties">

| Property | Description |
|---|---|
| `thrust-lbs` | Net thrust (lbf) |
| `total-impulse` | Cumulative actual impulse (lbf·s) |
| `fuel-flow-pps` | Propellant flow rate (lb/s) |
| `isp` | Current specific impulse |

  </Accordion>
</AccordionGroup>

### FGElectric — electric motor

**XML element:** `<electric_engine>`

Simple brushed electric motor. Throttle controls shaft power linearly from zero to the specified `<power>`. No battery model is included — the motor does not consume stored energy.

```xml
<electric_engine name="E-Motor">
  <power unit="WATTS"> 100000 </power>
</electric_engine>
```

| Parameter | Description |
|---|---|
| `power` | Maximum motor power |

The motor passes shaft power to an `FGPropeller` thruster. RPM is determined by the propeller load model.

## Thruster types

### FGPropeller — variable-pitch propeller

**Thruster definition file element:** `<propeller>`

Blade-element propeller model using tabular coefficients of thrust (Ct) and power (Cp) indexed by advance ratio J.

```xml
<propeller name="Clark-Y-76" version="1.1">
  <ixx>          1.67    </ixx>       <!-- slug·ft² -->
  <diameter unit="IN"> 76 </diameter>
  <numblades>    2       </numblades>
  <gearratio>    1.0     </gearratio>
  <minpitch>    12.0     </minpitch>  <!-- degrees -->
  <maxpitch>    45.0     </maxpitch>
  <minrpm>     500       </minrpm>
  <maxrpm>    2700       </maxrpm>
  <constspeed>   1       </constspeed>    <!-- 1 = constant-speed governor -->
  <reversepitch>-5.0     </reversepitch>
  <ct_factor>    1.0     </ct_factor>
  <cp_factor>    1.0     </cp_factor>

  <table name="C_THRUST" type="internal">
    <tableData>
        <!-- advance ratio J : Ct -->
        0.0   0.0782
        0.1   0.0759
        ...
    </tableData>
  </table>

  <table name="C_POWER" type="internal">
    <tableData>
        0.0   0.0512
        0.1   0.0498
        ...
    </tableData>
  </table>

  <!-- Optional Mach correction tables -->
  <table name="CT_MACH" type="internal">
    <tableData> ... </tableData>
  </table>
  <table name="CP_MACH" type="internal">
    <tableData> ... </tableData>
  </table>
</propeller>
```

<ParamField path="<sense>" type="1 | -1">
  Direction of rotation as seen from the cockpit: `1` = clockwise, `−1` = counter-clockwise. Specified in the **parent** `<thruster>` element, not the propeller file.
</ParamField>

<ParamField path="<p_factor>" type="number">
  P-factor (asymmetric disc effect) coefficient. Also specified in the parent `<thruster>` element.
</ParamField>

<ParamField path="constspeed" type="0 | 1">
  When `1`, a constant-speed governor varies pitch to hold RPM between `minrpm` and `maxrpm`.
</ParamField>

<Note>
  Set `version="1.1"` (or higher) in the propeller element to use the corrected gyroscopic moment sign. The absence of this attribute uses the legacy incorrect sign for backward compatibility.
</Note>

### FGNozzle — jet or rocket nozzle

**Thruster definition file element:** `<nozzle>`

Computes thrust from nozzle exit conditions for jet and rocket engines. Used with `FGTurbine` and `FGRocket` engine types via the `"direct"` thrust concept.

```xml
<nozzle name="direct">
  <area unit="FT2"> 1.0 </area>       <!-- nozzle exit area -->
  <pe>            10.9  </pe>         <!-- nozzle exit pressure (psi) -->
  <vacuum_thrust> 10000 </vacuum_thrust>  <!-- optional vacuum thrust (lbf) -->
  <thrust_table name="propulsion/thrust_ratio" type="internal">
    <tableData>
      <!-- Mach : thrust ratio relative to sea-level static -->
      0.0   1.000
      0.5   0.934
      1.0   0.917
    </tableData>
  </thrust_table>
</nozzle>
```

### FGRotor — helicopter rotor

**Thruster definition file element:** `<rotor>`

Blade-element rotor model for helicopters. Computes lift, drag, and flapping moments. Supports main rotors, tail rotors, and tandem configurations via `<controlmap>`.

```xml
<rotor name="main-rotor">
  <diameter unit="FT">      44.0  </diameter>
  <numblades>                4    </numblades>
  <gearratio>                9.17 </gearratio>
  <nominalrpm>             258    </nominalrpm>
  <minrpm>                   1    </minrpm>
  <maxrpm>                 516    </maxrpm>
  <chord unit="FT">          1.73 </chord>
  <liftcurveslope Xunit="1/RAD"> 5.7 </liftcurveslope>
  <twist unit="DEG">        -8.0  </twist>
  <hingeoffset unit="FT">    1.05 </hingeoffset>
  <flappingmoment unit="SLUG*FT2"> 678 </flappingmoment>
  <polarmoment unit="SLUG*FT2">   1050 </polarmoment>
  <inflowlag>                0.15 </inflowlag>
  <tiplossfactor>            0.97 </tiplossfactor>
  <maxbrakepower unit="HP">  20   </maxbrakepower>
  <gearloss unit="HP">        5   </gearloss>
  <controlmap> MAIN </controlmap>
  <groundeffectexp>   0.04 </groundeffectexp>
  <groundeffectshift unit="FT"> 4.5 </groundeffectshift>
</rotor>
```

**Control inputs** (read from the property tree at runtime):

| Property | Rotor type | Description |
|---|---|---|
| `propulsion/engine[n]/collective-ctrl-rad` | MAIN / TANDEM | Collective pitch |
| `propulsion/engine[n]/lateral-ctrl-rad` | MAIN | Lateral cyclic |
| `propulsion/engine[n]/longitudinal-ctrl-rad` | MAIN | Longitudinal cyclic |
| `propulsion/engine[n]/antitorque-ctrl-rad` | TAIL | Tail rotor collective (anti-torque pedal) |

**`<controlmap>` values:** `MAIN`, `TAIL`, `TANDEM`

## Tank types

Tanks are declared in the `<propulsion>` section and referenced by index from `<engine>` via `<feed>`.

```xml
<tank type="FUEL">
  <location unit="IN">
    <x>   0 </x>
    <y>  18 </y>
    <z>  -8 </z>
  </location>
  <capacity unit="LBS"> 114 </capacity>
  <contents unit="LBS">  57 </contents>
  <density unit="LBS/GAL"> 6.0 </density>
  <type> 1 </type>
</tank>
```

| Attribute / Element | Description |
|---|---|
| `type` attribute | `FUEL` or `OXIDIZER` |
| `<location>` | Tank CG location in structural frame |
| `<capacity>` | Maximum usable fuel weight |
| `<contents>` | Initial fuel weight |
| `<density>` | Fuel density (lb/gal). Default is auto-selected by `<type>`. |
| `<type>` | Fuel type code: `1` = mogas / avgas, `2` = Jet-A |

**State properties** at `propulsion/tank[n]/`:

| Property | Description |
|---|---|
| `contents-lbs` | Current fuel quantity (lbs) |
| `pct-full` | Fill fraction 0–1 |
| `fuel-consumption-rate-pps` | Burn rate (lb/s) |
| `empty` | `1` when tank is empty |
