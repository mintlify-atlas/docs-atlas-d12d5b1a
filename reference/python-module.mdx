The `jsbsim` package exposes a Cython-generated extension module (`_jsbsim`) plus a thin `__init__.py` re-export layer. All names below are importable directly from the top-level `jsbsim` namespace.

```python
import jsbsim

fdm = jsbsim.FGFDMExec(root_dir=jsbsim.get_default_root_dir())
```

## Module-level utilities

### get_default_root_dir

```python
def get_default_root_dir() -> str
```

Searches the Python site-packages directories for the bundled `jsbsim/aircraft/` data tree and returns the root path. Raises `IOError` if not found.

```python
root = jsbsim.get_default_root_dir()
# e.g. '/usr/local/lib/python3.11/site-packages/jsbsim'
```

### set_logger

```python
def set_logger(logger: FGLogger) -> None
```

Replaces the active logging backend. `logger` must be an instance of `FGLogger` or a subclass. The default at import time is `DefaultLogger`, which writes to `stdout`.

## FGFDMExec

The main simulation executive. Inherits from `FGJSBBase`.

```python
class FGFDMExec(root_dir: str, pm_root: FGPropertyManager = None)
```

<ParamField path="root_dir" type="str" required>
  Path to the JSBSim root directory (the one containing `aircraft/`, `engine/`, and `systems/` subdirectories). Pass `None` to use `get_default_root_dir()` automatically.
</ParamField>

<ParamField path="pm_root" type="FGPropertyManager">
  Optional existing property manager to attach to. Leave `None` in normal use.
</ParamField>

`FGFDMExec` supports property access via the subscript operator:

```python
vt = fdm["velocities/vt-fps"]          # read
fdm["fcs/elevator-cmd-norm"] = 0.05    # write
```

### Simulation loop

<AccordionGroup>
  <Accordion title="run() / run_ic()">

```python
def run(self) -> bool
```

Advances simulation by one `dT`. Returns `True` while the simulation should continue.

```python
def run_ic(self) -> bool
```

Applies initial conditions with `dt = 0`. Call once after loading a model and after every `reset_to_initial_conditions()`.

```python
fdm.load_model("c172r")
fdm.run_ic()
while fdm.run():
    print(fdm.get_sim_time())
```

  </Accordion>
  <Accordion title="reset_to_initial_conditions(mode)">

```python
def reset_to_initial_conditions(self, mode: int) -> None
```

Resets all models. `mode=0` is a plain reset; `mode=1` opens a new output file; `mode=2` skips the internal `run_ic()` call.

  </Accordion>
</AccordionGroup>

## FGPropertyNode

Represents a single node in the property tree.

```python
def get_name(self) -> str
```
Returns the node's simple (unqualified) name.

```python
def get_fully_qualified_name(self) -> str
```
Returns the full path from root, e.g. `"/velocities/vt-fps"`. This call traverses the tree and is slow; use it for debugging only.

```python
def get_node(self, path: str, create: bool = False) -> Optional[FGPropertyNode]
```
Navigates to a child node by relative path. Returns `None` if the node does not exist and `create` is `False`.

```python
def get_double_value(self) -> float
def set_double_value(self, value: float) -> bool
```
Read or write the node's numeric value.

```python
def get_attribute(self, attr: Attribute) -> bool
def set_attribute(self, attr: Attribute, state: bool) -> None
```
Check or set read/write attributes on the node using the `Attribute` enum.

## FGLogger

Abstract base class for custom log sinks. Subclass this to redirect JSBSim output.

```python
class FGLogger:
    log_level: LogLevel = LogLevel.BULK

    def set_level(self, level: LogLevel) -> None
    def file_location(self, filename: str, line: int) -> None
    def message(self, message: str) -> None
    def format(self, format: LogFormat) -> None
    def flush(self) -> None
```

Register with `jsbsim.set_logger(my_logger)`.

## Exceptions

| Exception | Inherits from | Raised when |
|---|---|---|
| `BaseError` | `RuntimeError` | General JSBSim errors |
| `TrimFailureError` | `BaseError` | Trim algorithm fails to converge |
| `GeographicError` | `BaseError` | Geographic computation error (e.g. invalid coordinates) |

```python
try:
    fdm.do_trim(1)
except jsbsim.TrimFailureError as e:
    print(f"Trim failed: {e}")
```
